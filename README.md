# Weather App (Swing + wttr.in)

A simple Swing-based weather app that fetches real-time weather data from wttr.in—no API key required.

## Features
- Displays temperature, humidity, wind speed, and conditions.
- Supports Celsius and Fahrenheit units.
- Built with Java 17 and Swing.

## How to Run
1. Clone this repository.
2. Run `mvn compile exec:java -Dexec.mainClass="WeatherAppSwingNoAPIKey"`.
3. Enter a city name (e.g., "London") and select units.

## Requirements
- Java 17+
- Maven
